# Gamestation_SC95F8736

基于 **SC95F8736 8 位单片机** 的迷你游戏机嵌入式工程，适配 **Keil C** 开发环境。项目实现了“飞机大战”等基础游戏功能，并可扩展闹钟等实用功能；同时包含 SPI 通信测试、底层驱动、芯片配置、固件库及相关设计文档，可作为 SC95F8736 外设开发与小型嵌入式系统设计的参考工程。

## 项目简介

本项目围绕 SC95F8736 单片机进行嵌入式应用开发，主要包含：

- 基于单片机的迷你游戏机功能设计；
- “飞机大战”游戏逻辑实现；
- 闹钟等基础功能扩展；
- SPI 总线通信功能测试与验证；
- GPIO、SPI 等外设底层驱动封装；
- Keil C 工程结构、芯片配置文件与固件库集成。

项目采用较清晰的分层结构，包含芯片配置层、固件库层、驱动层和用户应用层，便于后续功能扩展、代码维护和同系列 MCU 项目移植。

## 项目目录结构

```text
Gamestation_SC95F8736/
├── CFG/
│   └── SC95F8736/          # SC95F8736 芯片底层配置文件，如时钟、引脚等
├── Keil_C/                 # Keil C 完整工程目录
│   ├── Bank1/              # 程序存储分区 1
│   ├── Bank2/              # 程序存储分区 2
│   ├── Bank3/              # 程序存储分区 3
│   ├── Drivers/            # 外设驱动层，如 GPIO、SPI 等
│   ├── FWLib/              # 芯片标准固件库
│   ├── List/               # 编译生成的列表文件
│   ├── Output/             # 编译输出目录，包含 .hex / .bin 等固件文件
│   ├── Project/            # Keil 工程配置文件
│   └── User/               # 用户应用层代码，如主程序、游戏逻辑、SPI 测试逻辑
├── Document.pdf            # 项目相关文档，如芯片说明、测试方案、电路说明等
├── readme.txt              # 简易说明文档
├── spi_test.cgen           # SPI 测试代码生成配置文件
├── spi_test.fig            # SPI 通信时序图 / 硬件设计图
└── spi_test_01.fig         # 辅助设计图纸
```

## 开发环境

- **开发工具**：Keil C
- **主控芯片**：SC95F8736 8 位 MCU
- **开发语言**：C
- **输出文件**：`.hex` / `.bin`
- **烧录方式**：使用与 SC95F8736 适配的烧录器或仿真器，具体以实际硬件平台为准

## 快速开始

### 1. 克隆仓库

```bash
git clone git@github.com:WU-ss/Gamestation_SC95F8736.git
```

如果你的 GitHub 仓库名为 `Gamestation`，可以使用：

```bash
git clone git@github.com:WU-ss/Gamestation.git
```

### 2. 打开 Keil 工程

打开 Keil C，选择：

```text
Project → Open Project
```

然后进入：

```text
Keil_C/Project/
```

选择对应的 Keil 工程文件，例如 `.uvproj` 或 `.uvprojx`。

### 3. 编译工程

在 Keil 中点击：

```text
Build
```

工程已集成基本固件库和底层驱动代码，正常情况下无需额外配置即可编译。

### 4. 获取固件文件

编译完成后，可在以下目录中查看输出文件：

```text
Keil_C/Output/
```

常见输出包括：

```text
.hex
.bin
```

### 5. 烧录运行

使用适配 SC95F8736 的烧录工具，将生成的固件文件烧录到单片机中。上电后，系统将运行对应的游戏机程序或 SPI 测试程序。

## 核心功能

### 游戏机功能

- 实现“飞机大战”等基础游戏逻辑；
- 支持按键输入、显示刷新与游戏状态切换；
- 可扩展闹钟、计时器等实用功能；
- 适合作为小型单片机游戏机或嵌入式交互系统原型。

### SPI 通信测试

- 支持 SPI 总线通信功能验证；
- 可用于 SPI 主机模式下的数据收发测试；
- 支持外接 SPI 设备的联调，如显示屏、传感器、存储芯片等；
- 可作为后续外设驱动开发的基础模板。

### 工程结构

- **CFG**：芯片基础配置；
- **FWLib**：标准固件库；
- **Drivers**：外设驱动层；
- **User**：用户应用逻辑；
- **Output**：编译输出文件。

## 注意事项

1. 烧录前请确认芯片型号、供电电压、晶振配置和引脚连接是否与工程配置一致。
2. 若编译失败，请优先检查 Keil 版本、工程路径以及芯片支持包是否完整。
3. 若 SPI 通信异常，请检查 CPOL / CPHA、片选信号、通信速率和硬件连线。
4. 若游戏显示或按键异常，请检查显示屏接口、按键 IO 配置以及中断/定时器设置。
